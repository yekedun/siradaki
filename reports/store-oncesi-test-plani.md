# Store Oncesi Test Plani

Bu dosya genel referanstir. Ayrintili ve session bazli gorevler `reports/store-test-plan/` altindadir. Claude Code'a parca parca vermek icin once `reports/store-test-plan/index.md` dosyasini kullan.

Ana kapsam:

- Repo, lint, typecheck, test ve build kontrolleri
- Supabase migration, database type sync, appointment constraint ve trigger kontrolleri
- Shared slot algoritmasi, Deno import uyumu ve timezone/DST testleri
- Edge function auth, CORS, validation, token ve conflict testleri
- Web booking, admin approve/reject ve invite deep link testleri
- Secret, authorization, token, input validation ve log guvenligi
- Paralel booking, invite accept race ve widget concurrency testleri
- Availability/booking/push load testleri
- iOS TestFlight ve Android internal test gercek cihaz kontrolleri
- App Store Connect ve Google Play Console metadata/privacy/data safety kontrolleri
- In-app account deletion, Google web deletion URL ve privacy policy kontrolleri
- Final release gate: Hazir / Hazir degil karari
